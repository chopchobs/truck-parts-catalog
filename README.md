# 🚛 C.RUAMCHANG (อู่ ช.รวมช่าง) - Truck Services & Parts Catalog

[![Next.js](https://img.shields.io/badge/Next.js-14+-black?style=for-the-badge&logo=next.js)](https://nextjs.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)](https://tailwindcss.com/)
[![Supabase](https://img.shields.io/badge/Supabase-3ECF8E?style=for-the-badge&logo=supabase&logoColor=white)](https://supabase.com/)
[![Vercel](https://img.shields.io/badge/Vercel-000000?style=for-the-badge&logo=vercel&logoColor=white)](https://vercel.com/)

เว็บไซต์อย่างเป็นทางการของ **อู่ ช.รวมช่าง** (อ.หล่มสัก จ.เพชรบูรณ์) บริการรับซ่อมรถบรรทุก รถพ่วง รถสิบล้อ และแพลตฟอร์มแคตตาล็อกจำหน่ายอะไหล่รถใหญ่แบบครบวงจร 

ออกแบบด้วยดีไซน์ที่ทันสมัย (Modern UI) ใช้งานง่าย รองรับการแสดงผลทุกหน้าจอ (Responsive Design) พร้อมระบบจัดการสินค้าและบริการหลังบ้าน

🌐 **Live Demo:** [https://cruamchang.vercel.app](https://cruamchang.vercel.app)

---

## ✨ Features (ฟีเจอร์หลัก)
- **Hero & Landing Page:** หน้าแรกดึงดูดสายตา พร้อมบอกพิกัดและจุดเด่นของอู่ชัดเจน
- **Services Showcase:** นำเสนอบริการซ่อมบำรุงต่างๆ (ช่วงล่าง, เบรก, เกียร์, แชสซี ฯลฯ) พร้อมเอฟเฟกต์การใช้งานที่ลื่นไหล
- **Product Catalog:** ระบบแสดงสินค้าและอะไหล่รถบรรทุก
- **Responsive Design:** แสดงผลได้สวยงามทั้งบน Mobile, Tablet และ Desktop
- **SEO Optimized:** ปรับแต่ง Metadata และ Favicon เพื่อการค้นหาบน Google ที่ดีขึ้น

---

## 🛠 Tech Stack (เทคโนโลยีที่ใช้)
- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Authentication:** NextAuth.js
- **Database / Backend:** Supabase
- **Deployment:** Vercel

## 🚀 Getting Started (วิธีรันโปรเจกต์ในเครื่อง)

หากต้องการดาวน์โหลดโค้ดไปพัฒนาต่อ สามารถทำตามขั้นตอนด้านล่างนี้ได้เลยครับ:

**1. Clone the repository**
```bash
git clone [https://github.com/chopchobs/truck-parts-catalog.git](https://github.com/chopchobs/truck-parts-catalog.git)
cd truck-parts-catalog 
```
**2. Install dependencies**
```bash
npm install
```
**3. Set up Environment Variables**
```bash
DATABASE_URL="your_supabase_db_url"
DIRECT_URL="your_supabase_direct_url"
NEXTAUTH_SECRET="your_nextauth_secret"
NEXTAUTH_URL="http://localhost:3000"
```
**4. Run the development server**
```bash
npm run dev
```
